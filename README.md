## Problem Statement
Loan approval is a critical process in the banking sector.
Manual evaluation is time-consuming and often lacks transparency.
There is a need for an automated and explainable AI-based system
to assist banks in loan approval decisions.

---

## Solution Overview
This project implements a Machine Learning and AI-based loan approval system that:
- Predicts loan approval status
- Uses Explainable AI (SHAP) to justify decisions
- Generates human-readable explanations
- Performs basic fairness checks
